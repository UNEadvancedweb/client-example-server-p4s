Examples server

This is a bare-bones Play for Scala install, with the routes set to serve everything from the /public folder.

Put your examples into subdirectories, and access them accordingly. Eg: 

    http://localhost:9000/myFirstExample/index.html

Please note that unlike Apache, this server does not auto-detect if your URL points to a directory instead of a file. I have however made it so that anything that ends with a / redirects to index.html.

So, this will not produce 404 as it points to a directory but does not end /:

  • /myFirstExample
But this will work, redirecting to the index.html file:
  • /myFirstExample/
And this will work too:
  • /myFirstExample/index.html
